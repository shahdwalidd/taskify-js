# 📝 Taskify JS

A simple and elegant **To-Do List** web app built using **HTML**, **CSS**, and **JavaScript**.  
You can **add, edit, delete**, and **mark tasks as complete**, with all your data saved locally using **localStorage**.

---

## 🚀 Features
- ➕ Add new tasks  
- 🖊️ Edit existing tasks  
- ✅ Mark tasks as done or undone  
- 🗑️ Delete tasks  
- 💾 Auto-save tasks using `localStorage`

---

## 🧠 Technologies Used
- HTML5  
- CSS3  
- Vanilla JavaScript
